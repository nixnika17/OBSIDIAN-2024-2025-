Проєкція в повітрі без екрану з сенсорним управлінням — наукова формулювання

1. Мета
Створити видиме тривимірне зображення в повітряному об’ємі (без фізичного екрана) з інтерактивністю рівня «дотик/жест» і тактильним фідбеком.


2. Фізичні принципи візуалізації (варіанти)
• Краплинне/аерозольне розсіяння (вольюметрична проєкція): проектор світить у хмару дрібних частинок; видиме поле I(x,y,z) ≈ L·σ_s·n(x,y,z), де L — світловий потік, σ_s — перетин розсіяння частинок, n — концентрація частинок. Плюс: простота; мінус: чутливість до освітлення й обмежена роздільність.
• Лазерна індукована плазма: фокусований імпульсний лазер утворює плазмові точки в повітрі (локальні випромінюючі вокселі). Плюс: видиме в чистому повітрі; мінус: енергетично та безпеково проблематично.
• Оптичні світлові поля / голографія: створення об’ємного зображення через інтерференцію/просторове кодування променів — потребує складної апаратури і часто непридатне для «вільного» повітря.


3. Трекінг та інтерфейс дотику
• 3D-трекінг: Time‑of‑Flight або стерео‑глибина + алгоритми виявлення кінчиків пальців (наприклад, depth + MediaPipe‑подібні підходи) для виявлення координат пальця P(t) в реальному просторі.
• Реєстрація «натиску»: модель детекції контакту: якщо |P(t) − V| < Δ та d/dt(|P−V|) < −v_threshold → «натиск», де V — цільова воксельна позиція.
• Тактивний фідбек: ультразвукові фазовані масиви для створення фокусованого акустичного тиску в точці (mid‑air haptics) або короткі повітряні імпульси / плазмові імпульси для відчуття «кліку».


4. Інженерні обмеження (ключові)
• Яскравість/контраст: без частинок або плазми світло не сприймається; потрібні або великі L, або робоче середовище з частинками; видимість ∝ L·σ_s·n.
• Роздільність: для плавного зображення потрібна висока щільність вокселів — ресурсно дорого.
• Безпека: лазери та плазма — ризик для очей; ультразвук/високі інтенсивності можуть призвести до дискомфорту.
• Латентність: інтерфейс має <50 ms загальної затримки, щоб виглядати «чутливим».


5. Рекомендований мінімальний прототип (гіпотетично, економно)
Комбінація: короткофокусний яскравий проектор + контрольована аерозольна хмара + depth‑камера для трекінгу + простий тактильний модуль (повітряний імпульс або ультразвук). Це дає видиме інтерактивне вольюметричне відображення з прийнятним UX в темному/напівтемному середовищі.
