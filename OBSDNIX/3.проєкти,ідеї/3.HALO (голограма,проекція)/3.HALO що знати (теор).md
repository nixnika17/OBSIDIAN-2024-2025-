Якщо коротко: оптика + фізика + комп’ютерний зір + електроніка + програмування + 3D-графіка + безпека + математика.
Тобі не обов’язково бути експертом в усьому одразу, але кожен блок треба хоча б на рівні «розумію, як це працює і як поєднати».



---

1. [ ] Фізика світла та оптики

Принципи відбиття, заломлення, розсіяння світла.

Вольюметрична оптика: як світло взаємодіє з частинками у повітрі.

Лазери та плазма: як створюються локальні світлові точки.

Інтерференція та голографія (базовий рівень, щоб розуміти принцип).


Навички: розрахунок яскравості, роздільності, інтенсивності лазерів.


---

2. Комп’ютерний зір та трекінг

Time-of-Flight камери, стерео-глибина, LIDAR.

Виявлення пальців та жестів (MediaPipe, OpenCV).

Програмування алгоритмів взаємодії 3D-координат пальців з «вокселями» голограми.


Навички: Python/C++ для обробки відеопотоку, бібліотеки для 3D-трекінгу.


---

3. Електроніка та сенсорні системи

Робота з датчиками руху, ультразвуковими фазованими матрицями для тактильного фідбеку.

Управління імпульсними пристроями (соленоїди, повітряні форсунки).

Інтеграція сенсорів з контролером/ПК.


Навички: Arduino / Raspberry Pi / мікроконтролери, базова схемотехніка.


---

4. Програмування та інтеграція

Системна інтеграція: обробка сигналів з сенсорів → алгоритм «натискання» → відображення світла → тактильний фідбек.

Реалтайм обробка (низька затримка, <50 ms).

Можна використовувати Python + C++ + Unity/Unreal для прототипу.



---

5. 3D-графіка та візуалізація

Робота з 3D-простором (вокселі, координати).

Вольюметричні рендери (як об’ємна голограма «малюється» в повітрі).

Контроль кольору, яскравості та відтінків у вокселях.



---

6. Безпека та матеріалознавство

Лазери і плазма — техніка безпеки.

Робота з дрібними частинками (туман, аерозоль).

Ультразвук і фізіологічні ефекти.



---

7. Математика

Лінійна алгебра (вектори, матриці для 3D-координат).

Геометрія та тригонометрія (для проєкції на об’єм).

Базова фізика: енергія, світловий потік, інтенсивність.
